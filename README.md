# Python USSD & Linguistics Project
### KWASU 300-Level Portfolio

This repository contains Python scripts developed for my undergraduate studies in Linguistics at Kwara State University.

## 📱 USSD Simulation (`ussd_assignment.py`)
A functional script simulating mobile network menus. It handles phone number prefixes and simulates data bundle purchases (MTN/MoMo).

## 📐 Linguistics Formulas (`formula.py`)
Mathematical scripts used for calculations relevant to phonetic studies and acoustic analysis.

## 🚀 Goals
I am building this portfolio to transition into **Computational Linguistics** for my Master's studies in Japan.
